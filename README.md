Descrição do Projeto
Este é um aplicativo de console desenvolvido em **TypeScript** que funciona como um Gerenciador de Biblioteca Pessoal. O sistema utiliza o conceito de **arrays paralelos** para gerenciar informações como títulos, autores, anos de publicação e status de leitura[cite: 1]. 

O projeto consolida conhecimentos em:
* Manipulação de Arrays (`map`, `filter`, `reduce`, `forEach`).
* Tipagem estrita com TypeScript.
* Controle de fluxo e funções tipadas.
* Histórico de versionamento com Git.

Instruções de Execução

Pré-requisitos
* Node.js instalado (versão 18 ou superior).
* Gerenciador de pacotes npm.

Passo a Passo
1. **Instalar dependências:**
   No terminal, dentro da pasta do projeto, execute:
   ```bash
   npm install
   

2. **Compilar e Executar o projeto:**
   Para rodar a demonstração final configurada no `package.json`:
   ```bash
   npm run dev
   
```

> **Nota:** O comando `npm run dev` executa o compilador TypeScript (`tsc`) e, em seguida, inicia o arquivo gerado na pasta `dist`[cite: 1].
